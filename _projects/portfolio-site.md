## Overview

A personal portfolio and blog built with [Jekyll](https://jekyllrb.com/) and hosted for free on GitHub Pages. The site features a dark-mode design, responsive layout, a projects showcase, and a blog.

## Features

- Responsive, dark-themed design
- Project collection with detail pages
- Blog with posts organized by date
- Simple navigation and clean typography
- Hosted on GitHub Pages — no server needed

## How it was built

The site uses Jekyll's collections feature to manage projects and posts separately. All layouts are hand-crafted in plain HTML/CSS without any third-party UI frameworks, keeping the footprint minimal.

## Running locally

```bash
bundle install
bundle exec jekyll serve
```

Then open `http://localhost:4000` in your browser.
