# havriutkin.github.io

Personal site built with [Jekyll](https://jekyllrb.com/) and hosted on [GitHub Pages](https://pages.github.com/).

**Live site:** https://havriutkin.github.io

## Features

- Dark-themed landing page with hero section
- **Projects** showcase with individual detail pages
- **Blog** with posts organized by date and tags
- RSS feed (`/feed.xml`)
- Responsive, mobile-friendly design

## Local development

```bash
bundle install
bundle exec jekyll serve
# Open http://localhost:4000
```

See [instructions.md](instructions.md) for a complete guide on:
